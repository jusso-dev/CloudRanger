# Example executive reports

These PDFs are generated from deterministic dummy scan data so prospective users can see the CloudRanger executive report format without access to a cloud account. They do not describe real environments and must not be used as evidence of compliance.

- [AWS example report](aws-example-executive-report.pdf)
- [Azure example report](azure-example-executive-report.pdf)
- [GCP example report](gcp-example-executive-report.pdf)

Regenerate them after building the CLI with:

```bash
pnpm examples:reports
```
